# 第五章

黑暗，冰冷，積水過膝。

大樓十四樓的主機備份層，此時已被颱風灌入的積水淹沒。一排排巨大的黑色伺服器機櫃，如同黑暗中聳立的墓碑，靜靜地矗立在冰冷的積水中。

在剛才從空調管道瘋狂下滑的混亂與漆黑中，陸言與林晚走散了。

「沙沙……沙沙……」

陸言握著手中的手動發電模擬對講機，腰部以下冰冷徹骨。突然，對講機的喇叭裡傳來了一陣刺耳的雜音，緊接著，是林晚急促而微弱的呼救聲：

「陸言……你在哪裡？我被困在 B 區伺服器室了……一個沉重的機櫃倒下來壓住了我的左腿……積水一直在漲，已經到我的胸口了……快來救我……」

電話那頭林晚的喘息聲與痛苦的呻吟聲，真實得讓陸言心如刀割。

然而，幾乎在同一時間——

在幾十公尺外、隱蔽在 C 區變壓器室上方的林晚，手中的對講機也傳來了陸言低沉、沉穩的聲音：

「林晚，我找到了安全通道的備用出口鐵門，就在 C 區排風口。門鎖生鏽了，我需要你把剛才在配電箱撿到的鐵撬拿過來，我們合力撬開它，快點過來！」

兩道聲音，在漆黑寂靜、水聲嘩啦的十四樓空間裡，交織成了一張致命的心理陷阱網。

「不對……」

林晚死死捂住嘴，不讓自己的呼吸聲傳出。她看著身旁的積水，大腦飛速運轉。如果對講機那頭的「陸言」是真的，那他現在應該在瘋狂尋找被壓在機櫃下的自己才對，怎麼會先跑去撬門？

而此時，身處 A 區水坑中的陸言，也緩緩閉上了雙眼。他按下了對講機的通話鍵：

「林晚，你聽得到嗎？我現在考你一個聲學常識，三年前小喬潛水事故遇難時，海水的平均聲速是多少？回答我，我馬上過去！」

對講機那頭沉默了兩秒，隨即傳來了「林晚」快速、精確的回答：

「在標準大氣壓下，水溫二十度，海水的聲速是一千五百三十一公尺每秒。陸言，你瘋了嗎？都什麼時候了你還問這個！積水已經到我的脖子了……救我！」

這個回答，精確得如同教科書。

「一千五百三十一公尺。」陸言握著對講機的手，緩緩垂了下來，嘴角泛起一抹徹骨的冷笑。

「回答得太完美了，林晚。但你忘了，真正的林晚是認知心理學家，她根本記不住這些枯燥的物理聲速數據！更重要的是——」

陸言那嚴苛的聽覺過敏，此時在對講機的背景音裡，聽出了一個致命的破綻：

「你說你被壓在機櫃下，積水已經到了你的脖子，你在痛苦地掙扎。但在對講機的背景音裡，除了你顫抖的嗓音，**居然沒有任何一絲水花拍打伺服器鐵門的漣漪聲與回音**！」

那聲音太過純淨、乾淨得像是在高規格錄音室裡錄製的語音樣本。

這根本不是林晚，而是擬態鬼讀取了陸言大腦記憶中關於林晚的知識特徵，強行在對講機電磁波裡合成出的虛假語音！

與此同時，林晚也用同樣的方法，在對講機裡向那個「陸言」提出了一個「記憶偏差測試」——她故意問了陸言一個三年前關於小喬遺體搜尋時不復存在的細節，而電話那頭的擬態鬼，順著她設定的錯誤記憶完美地附和了。

「它在騙我們。」林晚在黑暗中咬緊牙關。

兩顆高智商的大腦，在這一刻達成了跨越空間的絕對默契。

他們沒有掛斷對講機，而是順水推舟，一邊用虛假的答覆敷衍著對講機那頭的擬態鬼，一邊利用對講機信號源發射的微弱電磁波強度，在主控電腦的 3D 地圖上進行反向定位。

「陸言，我把鐵撬扔在 C 區的高壓配電室了，你去拿……」

「好，我現在就進去……」

兩人在心智與算法的雙向博弈下，一步步將那兩道對講機背後的虛假聲音，強行引導向了十四樓中央那間裝有厚重雙層防爆鐵門、且通有殘餘高壓電的「主電阻測試室」！

「就是現在！」

陸言與林晚幾乎在同一秒內，從 A 區與 C 區的暗影中暴起，兩人在變阻器室門前重逢，合力將厚重無比的防爆鐵門「轟」地一聲關上，並用鐵撬死死卡住了轉輪鎖。

防爆鐵門後，頓時傳來了兩聲非人、高頻、極度刺耳的尖叫聲。那兩道聲音在尖叫中瘋狂地切換著陸言、林晚、陸小喬、阿傑的嗓音，最後化為一陣「咔咔」的指甲逆向抓撓鐵門的刺耳摩擦聲。

陸言與林晚靠在鐵門上，看著彼此溼漉漉、狼狽不堪的模樣，大口大口地喘息著。

「我們成功了？」林晚看著他。

「這只是其中兩個複製品。」陸言握緊了手電筒，看著周圍漸漸被白霧填滿的走廊，神色依舊無比凝重，「大樓的電壓還在下降。當備用電源徹底耗盡、整座大樓歸於絕對無聲的黑暗時……真正的獵殺，才會開始。」
